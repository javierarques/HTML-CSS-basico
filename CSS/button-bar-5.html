Reproduce the behaviour of the Mac OS X buttons using CSS animations.

To achieve this effect, create a 'bounce' animation at Animate.css using the option 'Create custom build', selecting only the effects 'Attention seekers' and clicking 'Build it'. In the CSS file add this animation to the selector li:target a (selects links that are clicked)

Inside the downloaded .css file, there are different animations for Firefox, Webkit and IE. Each one has a certain number of keyframes within the animation. Add the following:

  • animation-direction: alternate
  • iterations: 2
  • duration: 0.5 seconds

Next, link this .css in the head of the html page.

If in time, try any other animation like 'tada'.
